# 오늘의 적립 카드 디자인 원복 리뷰

## 리뷰 결과

- 큰 결함 없음.

## 확인한 점

- 요청의 핵심은 기존 카드들을 새 다크 디자인으로 맞추는 것이 아니라 `오늘의 적립` 카드를 기존 홈 카드 문법에 맞추는 것이므로, 홈 전체 다크 오버라이드를 제거했다.
- `이번 2주 변동비`, `이번 달 고정비`, `Dev Ideas`에는 별도 다크 카드 스타일이 남지 않도록 했다.
- `오늘의 적립` 카드의 게이지도 홈의 기존 보라 그라데이션 토큰을 사용하도록 되돌렸다.
- CSS cache bust가 `index.html`과 `style.css` 양쪽에 반영됐다.

## 검증

- `npm.cmd run verify` 통과.
- `npm.cmd run pages:build` 통과.
- `_site` 산출물에서 새 cache bust와 `오늘의 적립` 카드 스타일 확인.

## 운영 확인

- 배포 후 갱신 예정.

## 잔여 리스크

- 운영 홈 화면에서 실제 로그인 데이터 로딩 후 카드 배경과 게이지 색상을 다시 확인해야 한다.
