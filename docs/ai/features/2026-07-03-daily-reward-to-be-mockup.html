앱 화면 변경점

  • 히어로카드는 유지합니다. 상단 예산 진행률은 현재 홈의 기준 앵커라 그대로 둡니다.
  • `오늘의 적립` 카드에는 오늘 고른 카드와 추가 적립만 먼저 보여줍니다.
  • 목표템은 대표 목표 1개를 우선 노출하고, 나머지는 작은 갤러리로 보조합니다.
  • 실패, 경고, 벌점 문구 대신 기본 적립, 회복, 쉬어가기권 언어를 사용합니다.

보상 레이어 반영

  • 가변보상: 투명한 오늘 보너스, 집중 시간 카운트다운, 작은 보너스 미션.
  • 자율성: 와인/재료/여행 중 오늘의 집중 선택, 쉬어가기권 노출.
  • 유능감: 연속 적립, 실버 단계, 지난달 대비 이번달 미니 그래프.
  • 서사: 목표 이미지와 진행률을 포인트 행보다 더 구체적인 물건으로 연결.
