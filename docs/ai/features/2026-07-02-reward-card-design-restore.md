# 오늘의 적립 카드 디자인 원복 계획

## 요청

- 방금 확대 적용한 홈 전체 다크 디자인을 원복한다.
- 원복된 기존 홈 디자인에 맞게 `오늘의 적립` 카드만 다시 변형한다.

## 결정

- 보상 산식 설정 기능은 유지한다.
- 홈의 `이번 2주 변동비`, `이번 달 고정비`, `Dev Ideas`는 기존 흰 카드/기존 게이지 스타일로 되돌린다.
- `오늘의 적립` 카드는 기존 디자인 시스템에 맞춰 흰 카드, 기존 텍스트 색상, 기존 보라 게이지(`--grad-bar`)를 사용한다.
- 캐시 문자열은 `20260702-reward-card-restore`로 갱신한다.

## 실행 슬라이스

- 수정 파일: `styles/60-urge.css`, `style.css`, `index.html`, 문서.
- 수정하지 말 것: 산식 계산, 설정 저장, 포인트 이름, 포인트 차감 장부.
- 검증: `npm.cmd run verify`, `npm.cmd run pages:build`, 운영 홈에서 기존 카드들은 흰색으로 돌아가고 `오늘의 적립` 카드도 흰 카드 체계로 보이는지 확인한다.
